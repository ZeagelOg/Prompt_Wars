# 🏟️ VenueFlow

[![Lighthouse Score](https://img.shields.io/badge/Lighthouse-90%2B-brightgreen)](#)
[![Test Coverage](https://img.shields.io/badge/Coverage-70%25%2B-blue)](#)

VenueFlow is a high-performance Progressive Web App (PWA) designed to revolutionize the attendee experience at large-scale sporting venues. By leveraging Google Maps Platform and Firebase, it provides real-time crowd insights and indoor navigation.

## 🚀 Vision
To eliminate the friction of large-scale events through predictive analytics and intelligent rerouting.

## 🏗️ Architecture
- **Frontend:** React + TypeScript + Tailwind CSS (Vite)
- **Backend:** Firebase (Firestore, Auth, FCM)
- **Maps:** Google Maps JS API (Heatmap & Overlay Layers)
- **CI/CD:** GitHub Actions -> Cloud Run
- **Testing:** Jest (Unit) + Playwright (E2E)
